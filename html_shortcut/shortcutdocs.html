shortcut for html

> Documentation

  • type: ! tab

    loads basic frameset of html file

  • type : ul>li*n

    give n number of li inside ul

  • press: shift+alt+downarrow

    copies selected segment onto below line

  • press: alt + mouseclick

    do this at all the points where you want to make changes or type simultaneously

  • table>tr>th*3

    gives 3 column heading in a row
    heading1 heading2 heading3

  • tr>td*3

    gives 3 column data in a row
    data1 data2 data3

  • press: ctrl + /

    comments the selected para or line
